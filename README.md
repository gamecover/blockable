# Blockable

이 프로젝트는 [NHN 2026 해커톤](https://nan2026.nhn.com/) 참가를 목적으로 제작하는 웹 기반 게임입니다.

## 게임 소개

세계 최고의 대장장이인 플레이어가 직접 도구 주머니를 들고 던전에 도전하는 로그라이크 블록 배치 게임입니다.

무작위로 생성된 분기형 지도를 탐색하고, 전투에서는 주머니에서 뽑은 블록을 퍼즐 판에 배치해 몬스터와 싸웁니다. 전투와 이벤트에서 블록을 추가·삭제·변경하며 덱을 강화하고, 마지막 10층의 보스를 처치하는 것이 목표입니다.

## 핵심 요소

- 로그라이크 덱빌딩과 블록 배치 퍼즐을 결합한 턴제 전투
- 전투와 이벤트로 구성된 10층 분기형 지도
- 매 턴 블록 5개를 뽑고, 그중 최대 3개를 배치하는 전투 방식
- 블록의 모양·색상·조합에 따른 공격, 방어, 회복 및 추가 효과
- 전투 보상과 이벤트를 통한 블록 덱 구성 및 강화

## 기술 구성

- Web
- React
- Phaser

## 실행 방법

### 사전 준비

- [Node.js](https://nodejs.org/) 설치
- 터미널에서 프로젝트 최상위 폴더로 이동

### 최초 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

실행 후 터미널에 표시되는 로컬 주소를 웹 브라우저에서 열면 됩니다.

### 배포용 빌드

```bash
npm run build
```

## 개발 상태

현재 기획 및 프로토타입 개발 단계입니다. 전투 효과의 세부 수치, 블록 조합 공식, 지도 생성 알고리즘과 경제 밸런스는 플레이 테스트를 거쳐 확정할 예정입니다.

자세한 내용은 [기획안](./docs/GAME_DESIGN.md)을 참고해주세요.
